SEO-Sympa - Déclaration d'Accessibilité

Déclaration d'Accessibilité

SEO-Sympa s'engage à rendre son site web accessible à tous les utilisateurs, conformément aux standards d'accessibilité.

État de conformité

Ce site web est partiellement conforme aux règles pour l'accessibilité des contenus Web

Préparation de cette déclaration d'accessibilité

Cette déclaration a été préparée le 12 Juin 2025. Elle est basée sur :

  • Une auto-évaluation réalisée par l'équipe de développement de SEO-Sympa
  • Des tests utilisateurs avec des personnes en situation de handicap

Outils utilisés pour l'évaluation

  • Lecteurs d'écran : NVDA, VoiceOver
  • Tests de navigation clavier
  • Vérification du contraste des couleurs

Technologies utilisées

L'accessibilité de ce site web s'appuie sur les technologies suivantes :

  • HyperText Markup Language - HTML
  • Cascading Style Sheets - CSS
  • JavaScript
  • ARIA (Accessible Rich Internet Applications)
